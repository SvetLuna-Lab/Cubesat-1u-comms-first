# Changelog
All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning.

## [0.1.0] - 2025-12-28
### Added
- YAML-driven link budget calculator (FSPL, Prx, C/N0, Eb/N0).
- LEO Doppler estimate for frequency planning.
- Minimal beacon framing: sync + length + payload + CRC16-CCITT.
- Deterministic tests (pytest) and linting (ruff).
- GitHub Actions CI workflow.
- Baseline UHF config profile and S-band stub profile.
